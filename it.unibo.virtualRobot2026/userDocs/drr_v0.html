DDR Robot Controller Sprint1

Introduction

L'obiettivo del progetto è quello di costruire un sistema che comandi un DDR.

Requirements

  • Costruire un sistema software che comandi un Differential Drive Robot (DDR) in modo che, partendo dalla posizione iniziale HOME, il robot percorra il perimetro di una stanza.
  • Il sistema deve utilizzare il radar, in particolare:
    • Se il robot si sta muovendo e incontra un ostacolo a distanza inferiore a DMIN allora si ferma
    • Se il robot è fermo davanti ad un ostacolo allora ruota su sè stesso finchè non scompare

Requirement analysis

  • L'ambiente del robot è modellato con una griglia di celle, per comodità del sistema una cella è della grandezza minima per contenere il robot. 
    
record Stanza(int larghezza, int altezza) { }
  • Per interagire con il radar è necessara un'entità wrapper che astrae la funzionalità del radar già presente. Questo attore riceve continuamente la distanza rilevata e al passaggio dalla soglia DMIN emette opportuni eventi. 
    
QActor radar context ctxddr {
    State s0 initial {
        // rilevamento della distanza D (in cm)
        if( D < DMIN ) {
            emit radar : vicino(D)
        }
        else{
            emit radar : lontano(D)
        }
    }
}
    in alternativa, il radar potrebbe agire da semplice proxy, reinviando la distanza rilevata, in modalità push o pull al sistema che ne fa richiesta. Sarà allora compito del DDR stabilire se la distanza è inferiore alla soglia o meno, e agire di conseguenza.
  • Infine è necessario un'ulteriore attore che comanda il DDR secondo i requisiti specificati. Questo attore si occupa di muovere il DDR lungo il perimetro della stanza, e di reagire agli eventi emessi dal radar.
